This folder contains sample automation workflows exported from n8n.


1: **Lending Automation Workflow**
**Purpose:** Automates decision routing for a lending process.  
**Key Features:**
- Reads input data
- Branches into **Pending**, **Approved**, or **Rejected** logic
- Sends notifications or outputs based on status
- Prevents duplicate actions using internal checks

---

2: **Book Automation Workflow**
**Purpose:** Automates a content-processing or book-related workflow.  
**Key Features:**
- Parses input text or data
- Processes or transforms content
- Uses HTTP/API calls where needed
- Outputs structured results

---

3: **Web Scraping Automation**
**Purpose:** Demonstrates simple automated data extraction.  
**Key Features:**
- Fetches data via HTTP requests
- Cleans and formats the extracted data
- Outputs results for further automations
